Template Pattern

定义一个操作中算法的骨架，而将一些步骤延迟到子类中，在不改变算法结果的情况下重新定义它的步骤。

与strategy模式的区别:
Template Method 模式 适用于存在几个概念上相似，但不相同的过程。每个过程都是互相耦合的，因为他们与某个过程相关。
