#Install Package

国内安装非常慢，通常使用手动安装

  • 手动安装方法一：
    • cd ~/.atom/packages
    • git clone https://github.com/atom/vim-mode.git
    • cd vim-mode
    • npm install
  • 手动安装方法二：
    • apm install atom/vim-mode
